# 🌟 Plantilla - Coleccionador de Pociones | Motores de Videojuegos I

Bienvenido a la plantilla base del proyecto **Coleccionador de Pociones**, creada para la materia **Motores de Videojuegos I**.  
En este proyecto desarrollaremos un videojuego 2D usando **Unity** y **C#**.

## 🎮 ¿Qué haremos?

👉 Crear un repositorio en **GitHub** desde esta plantilla.  
👉 Clonarlo en tu computadora para comenzar el desarrollo.  
👉 Explorar la interfaz de **Unity** y configurar el proyecto.

## 🎯 Objetivo

- Familiarizarte con la interfaz de **Unity**
- Preparar el entorno de desarrollo para el proyecto

## 🧩 Requisitos

- **Unity**
- Editor de código recomendado: **Visual Studio** o **Sublime Text**
- Sistema operativo: **Windows**, **macOS** o **Linux**
- Plantilla: [S3-ColeccionandoPociones](https://github.com/JuegosTerva/S3-ColeccionandoPociones)
